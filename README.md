## Packages

``` react-native
React Navigation v5
Expo ActionSheet
KeyboardAwareScrollView
React Native Snap Carousel
React Native Maps
Axios
```

This project works on iOS and Android.

## Running the project

Clone the repository and install packages : 

```
npm install
```

When installation is complete :
```
expo start
```
